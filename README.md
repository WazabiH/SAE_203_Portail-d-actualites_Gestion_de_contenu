# Portail d'actualités – Front-Office & Administration

## Présentation

Ce projet a été réalisé dans le cadre de la SAE 2.03 du BUT Métiers du Multimédia et de l'Internet (MMI).

L'objectif était de développer une application web dynamique permettant la publication et la gestion d'articles. Le projet comprend un site public destiné aux visiteurs ainsi qu'un espace d'administration permettant de gérer les contenus.

Cette réalisation m'a permis de mettre en pratique le développement web full-stack avec PHP et MySQL, ainsi que les notions de gestion de contenu et d'administration de données.

## Fonctionnalités

### Front-office

* Consultation des articles publiés
* Affichage du détail d'un article
* Navigation entre les contenus
* Formulaire de contact
* Interface responsive

### Back-office

* Authentification administrateur
* Gestion des articles
* Gestion des auteurs
* Gestion des messages de contact
* Création, modification et suppression de contenus

## Installation

1. Cloner le dépôt :

```bash
git clone <url-du-repository>
```

2. Créer une base de données MySQL.

3. Importer le fichier SQL du projet.

4. Configurer les paramètres de connexion à la base de données.

5. Lancer le projet sur un serveur local (XAMPP, WAMP ou Laragon).
